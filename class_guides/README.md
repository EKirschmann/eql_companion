# Class guides

One curated Markdown file per class, injected into the Advisor and Gear
consults for every class in the character's trio. This is the home for
community playstyle wisdom the wiki/builds data cannot express — charm
tactics, spell-tiering priorities, pet gear rules, focus-farming routes.

`general.md` is special: cross-class mechanics and meta consensus,
loaded FIRST for every trio regardless of classes. `races.md`,
`stances_invocations.md`, and `rituals.md` are reference files loaded
only into the MAIN advisor consult (not the gear consult).

## Rules

- **Filename** = full class name, lowercase, spaces as underscores:
  `enchanter.md`, `shadow_knight.md`.
- The first ~2,600 characters feed the Advisor consult (~1,500 for the
  Gear consult) — put the most decision-relevant facts FIRST and keep
  files tight; prose past the cap is silently ignored.
- Cite your source and date at the top. Note the game version/era the
  advice comes from — after a patch, re-verify the load-bearing claims
  and update the date.
- Prefer falsifiable statements ("Whirl's stun outlasts its reuse timer
  at tier 7") over vibes ("Whirl is great"). The advisor quotes these to
  justify picks; wrong claims become wrong counsel.
- These files are yours to edit — they ship with defaults but nothing
  regenerates them.
