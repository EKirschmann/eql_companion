# Beastlord (EQL)

Baseline distilled from the eqlwiki.com class page (2026-07-21). Replace/extend with community wisdom as it appears; front-load decision-relevant facts.

- Role: hybrid melee + warder pet + sham-lite support. META WARNING
  (2026-07): currently rated underpowered — "stick with a shaman" — and
  7/7 notes buffed warders (mana to cast, better melee), so re-check.
- Warder: race-specific permanent companion, central to the class;
  dedicated pet heals (Keshuval`s L15, Herikol`s L27) and Spirit proc
  buffs (Lightning L13 / Blizzard L18 / Inferno L28, +75 DEX + attack).
- Support kit: slows (Drowsy L20, Sha`s Lethargy L50 — counts as a
  SLOWER for D3+ per meta), heals through Greater Healing, group regen
  (Spiritual Light L41), cures, invisibility.
- Stances: Balanced/Offensive/Evasive; invocations Recovery, Inversion,
  Over Channel, Spellblade.
- Trio: pet damage + slow + regen in one slot; endurance-hungry in
  Offensive/Evasive; multi-pet trios capped at ONE active pet.
