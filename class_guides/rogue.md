# Rogue (EQL)

Baseline distilled from the eqlwiki.com class page (2026-07-21). Replace/extend with community wisdom as it appears; front-load decision-relevant facts.

- Role: stealth melee assassin — best single-target burst via Backstab;
  chain armor, piercing.
- Backstab from ANY position with piercers (Chaotic Stab AA, free at
  L1); best backstab weapon works from Primary, Secondary, or Any slot
  (meta: keep a dagger in an Any slot).
- Sneak+Hide move-while-hidden (Innate Sneakiness AA, L1) — the
  strongest invisibility in the game; only top raid bosses see through
  it (meta). Evade (in-combat hide) sheds aggro (~65%).
- Poisons: 20+ types; ONE utility poison active (snare/stun/root/mana
  drain) while combat poisons stack.
- Stances: Balanced/Offensive/Striker/Evasive/Ranged; Unyielding
  invocation: 2x health regen + fear/mez/charm resistance.
- Trio: scouting + burst + lockpicking/disarm; +15 AGI/DEX class
  bonuses; meta: mote-farm engine when paired with ROG-heavy charm pets
  (see enchanter guide).
