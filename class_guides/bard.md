# Bard (EQL)

Baseline distilled from the eqlwiki.com class page (2026-07-21). Replace/extend with community wisdom as it appears; front-load decision-relevant facts.

- Role: jack-of-all-trades support — stacked songs give haste, run
  speed, resists, regen, mana song (stacks with Clarity), plus slows,
  lull, mez, charm (Solon`s Bewitching Bravura L39).
- Symphonic Aura AA (free, L1): auto-pulses qualifying songs (no mana,
  no cooldown, non-targeted) in the background; with /melody, ~9
  concurrent songs are practical.
- Landmarks: Selo`s Accelerando L5 (group run speed — the signature),
  Kelin`s Lament L8 (lull), Largo`s Melodic Binding L20 (AoE slow),
  Denon`s Desperate Dirge L43 (AoE damage).
- Stances Balanced/Offensive/Evasive; invocations incl. Recovery,
  Inversion, Over Channel, Inviolable. Instrument quality scales songs.
- Trio: counts as a SLOWER for D3+ content (meta), best group mobility,
  melee-capable. Caveat: DoTs break mez; busy piloting.
